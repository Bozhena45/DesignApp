Descripción detallada

Clase para el manejo de Fotos y insertarlos en ls bbdd rellenados con los parámetros correspondientes.

Esta clase sirve para poder crear una foto y poder insertarla dentro de la bbdd, tambien sirve para crear un JSON a partir de la libreria json = nlohmann::json, y este mensaje JSON es lo que se le enviará al cliente
